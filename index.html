Lyrics editor
Stage mode

• Stage shows only the live lyric window and transport controls.

• Set a scroll speed or BPM, press Start, and read your lines inside the glow.

• Drag the lyrics up or down during a song to fine‑tune position.

• Space = Start/Pause, R = Reset (desktop).

BPM mode tips: look up tempo on a BPM tool or metronome app; start with “4 lines = 1 measure”; if scrolling is too fast, increase lines per measure; if too slow, decrease it.

Edit mode

• Type or paste your lyrics into the editor.

• The stage lyric window mirrors the editor automatically.

• Use blank lines to separate verses, chorus, or bridge sections.

• Use Save to store this song to your device.
