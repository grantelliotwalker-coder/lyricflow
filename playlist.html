How to use the Editor

Adding lyrics

• Paste or type your lyrics into the editor below.

• Each line becomes one scrolling line on stage — keep them short enough to read at a glance.

• Use blank lines to create breathing space between verse, chorus and bridge.

Saving songs

• Tap Save As to name your song and add it to a playlist. This always creates a new song.

• Once a song is loaded, the button changes to Save and overwrites it directly.

• To duplicate a song, change its name before saving.

Loading & editing

• Tap Load to pick a saved song. It loads into the editor and onto the stage simultaneously.

• Edits you make are reflected live on stage — no need to reload.

Notes & settings

• The Notes button (top-right of editor) lets you store chords, cues or triggers with the song.

• In Edit Playlist, tap ℹ next to any song to set its scroll speed, font size and line height individually.

Playlists

• Songs are grouped by playlist name. Create a new playlist by typing a new name when saving.

• Use Edit Playlist to reorder or delete songs, rename songs, or rename the playlist.

• Use Export to save a backup file. Use Import to restore it on any device.
